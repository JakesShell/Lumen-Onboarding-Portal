# Lumen Onboarding Portal

## Overview

Lumen Onboarding Portal is a browser-based onboarding foundation built with Python and Flask.

The project is designed as a stronger future-facing base for a larger onboarding platform. It combines:

- A futuristic landing page
- A guided chatbot assistant
- A 10-question profile assessment
- A four-color onboarding style result system
- Multiple pages that support guided navigation and future expansion

Within the product concept:

- **LUMEN** is the main onboarding experience
- **AVIAN** is the structured intelligence and security layer behind the experience

## Why This Project Matters

This repository upgrades a loose chatbot concept into a more practical product foundation.

Instead of being only a simple conversational demo, it now acts as a base for future work in:

- Candidate experience
- Employee onboarding
- Guided workflow systems
- Communication-style interpretation
- Role-based onboarding pathways
- Internal HR or people-operations tooling

## Current Features

- Futuristic multi-page landing experience
- Built-in chatbot for page guidance
- 10-question assessment
- Four color profile categorization:
  - Red Driver
  - Blue Analyst
  - Green Supporter
  - Yellow Influencer
- Profile results page with:
  - strengths
  - onboarding style
  - manager guidance
  - score breakdown
- About page explaining LUMEN and AVIAN
- Pathways page for onboarding interpretation

## How To Run

1. Create or activate your virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Start the application:
   - `python .\app.py`
4. Open:
   - `http://127.0.0.1:5000`
