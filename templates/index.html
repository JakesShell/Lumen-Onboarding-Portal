The Four Color Groups

Red Driver

Direct, decisive, action-oriented.

Blue Analyst

Structured, logical, detail-focused.

Green Supporter

Reliable, steady, people-aware.

Yellow Influencer

Energetic, expressive, vision-led.

Why This Matters

This project is designed as a stronger base for a future onboarding platform. Instead of a generic chatbot, it becomes a structured product experience with pages, logic, results, and guided navigation.

Important Positioning

The four-color system should be used for communication style, learning preferences, and onboarding support. It should not be treated as a stand-alone hiring decision engine.
